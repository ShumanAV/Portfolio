Проект 1 - вебприложение для учета книг в библиотеке, работа с БД с помощью Spring JDBC Template.
Проект 2 - вебприложение для учета книг в библиотеке, работа с БД с помощью Spring Data JPA, а также отдельно на Spring Boot.
Проект 3 - имитация метеостанции, взаимодействие датчиков температуры и наличия дождя и сервера, передача результатов измерений на сервер, сервер и клиент (сенсоры), реализовано REST API.
