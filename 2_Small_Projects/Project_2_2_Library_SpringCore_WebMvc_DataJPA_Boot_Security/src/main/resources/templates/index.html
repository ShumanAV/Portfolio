Приложение для Библиотеки

В данном приложении можно просмотреть набор всех книг, которые есть в библиотеке, добавить новую книгу, изменить или удалить выбранную книгу.

Просмотреть находится ли книга на руках у посетителя или книга в данный момент свободна.

Можно назначить книгу выбранному человеку или освободить книгу, когда ее вернули.

При помощи поиска книг, можно найти нужную книгу/книги.
