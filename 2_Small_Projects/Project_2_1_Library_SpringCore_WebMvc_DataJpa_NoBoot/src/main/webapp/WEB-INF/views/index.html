Приложение для Библиотеки

Вы можете просмотреть набор всех книг, которые есть в библиотеке, добавить новую книгу, изменить или удалить выбранную книгу.

Можете просмотреть находится ли книга на руках у посетителя или книга в данный момент свободная.

Можете назначить книгу выбранному человеку или освободить книгу, когда ее вернули.
